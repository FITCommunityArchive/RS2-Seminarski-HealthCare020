# Healthcare020

<h4 >Scheduling medical examinations and visits, managing examination reports and refferals</h4>

### Prerequisites
* Installed Docker
* Android smarthpone or emulator

See the <a href="https://docs.docker.com/get-docker/">Docker website</a> for installation instructions.

### Build and run
1. Clone this repo

         git clone https://github.com/fahirmdz/HealthCare020.git
2. Build the images

         cd HealthCare020
         docker-compose build

3. Run

         docker-compose up
